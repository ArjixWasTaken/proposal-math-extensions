Math.constrain ( _x_, _lower_, _upper_ )

When the `constrain` function is called, the following steps are taken:

1. If no arguments are passed, return *NaN*; 1. If _x_ is *NaN*, return *NaN*. 1. If _lower_ is *NaN*, return *NaN*. 1. If _upper_ is *NaN*, return *NaN*. 1. If _x_ is one of *+∞*, *-∞*, return _x_. 1. Let _max_ be %Math_max%(_x_, _lower_). 1. Let _min_ be %Math_min%(_max_, _upper_). 1. Return _min_.

Math.map ( _x_, _inLow_, _inHigh_, _outLow_, _outHigh_ )

When the `map` function is called, the following steps are taken:

1. If no arguments are passed, return *NaN*; 1. If _x_ is *NaN*, return *NaN*. 1. If _inLow_ is *NaN*, return *NaN*. 1. If _inHigh_ is *NaN*, return *NaN*. 1. If _outLow_ is *NaN*, return *NaN*. 1. If _outHigh_ is *NaN*, return *NaN*. 1. If _x_ is one of *+∞*, *-∞*, return _x_. 1. Let _dividend_ be ((_x_ − _inLow_) × (_outHigh_ − _outLow_)). 1. Let _divisor_ be ((_inHigh_ − _inLow_) + _outLow_). 1. Let _quotient_ be (_dividend_ ÷ _divisor_). 1. Return _quotient_.

Math.fmap ( _x_, _inLow_, _inHigh_, _outLow_, _outHigh_ )

When the `fmap` function is called, the following steps are taken:

1. If no arguments are passed, return *NaN*; 1. If _x_ is *NaN*, return *NaN*. 1. If _inLow_ is *NaN*, return *NaN*. 1. If _inHigh_ is *NaN*, return *NaN*. 1. If _outLow_ is *NaN*, return *NaN*. 1. If _outHigh_ is *NaN*, return *NaN*. 1. If _x_ is one of *+∞*, *-∞*, return _x_. 1. Let _dividend_ be ((_x_ − _inLow_) × (_outHigh_ − _outLow_)). 1. Let _divisor_ be ((_inHigh_ − _inLow_) + _outLow_). 1. Let _quotient_ be (_dividend_ ÷ _divisor_). 1. Let _x32_ be the result of converting _quotient_ to a value in IEEE 754-2008 binary32 format using roundTiesToEven. 1. Let _x64_ be the result of converting _x32_ to a value in IEEE 754-2008 binary64 format. 1. Return the ECMAScript Number value corresponding to _x64_.

Math.radians ( _degrees_ )

When the `radians` function is called, the following steps are taken:

1. If no arguments are passed, return *NaN*; 1. If _degrees_ is one of *NaN*, *+∞*, *-∞*, return _degrees_. 1. Let _radians_ be (_degrees_ × (π ÷ 180)). 1. Return _radians_.

Math.degrees ( _radians_ )

When the `degrees` function is called, the following steps are taken:

1. If no arguments are passed, return *NaN*; 1. If _radians_ is one of *NaN*, *+∞*, *-∞*, return _degrees_. 1. Let _degrees_ be (_radians_ × (180 ÷ π)). 1. Return _degrees_.

Number.RAD_TO_DEG

The value of Number.RAD_TO_DEG is (180 ÷ π), which is approximately 57.29577951308232.

This property has the attributes { [[Writable]]: *false*, [[Enumerable]]: *false*, [[Configurable]]: *false* }.

Number.DEG_TO_RAD

The value of Number.DEG_TO_RAD is (π ÷ 180), which is approximately 0.017453292519943295.

This property has the attributes { [[Writable]]: *false*, [[Enumerable]]: *false*, [[Configurable]]: *false* }.
