Math.constrain ( _x_, _lower_, _upper_ )

When the `constrain` function is called, the following steps are taken:

1. If any argument is *NaN*, return *NaN*. 1. If _x_ is one of *+∞*, *-∞*, return _x_. 1. Let _max_ be %Math_max%(_x_, _lower_). 1. Let _min_ be %Math_min%(_max_, _upper_). 1. Return _min_.

Math.map ( _x_, _inLow_, _inHigh_, _outLow_, _outHigh_ )

When the `map` function is called, the following steps are taken:

1. If any argument is *NaN*, return *NaN*. 1. If _x_ is one of *+∞*, *-∞*, return _x_. 1. Let _dividend_ be ((_x_ − _inLow_) × (_outHigh_ − _outLow_)). 1. Let _divisor_ be ((_inHigh_ − _inLow_) + _outLow_). 1. Let _quotient_ be (_dividend_ ÷ _divisor_). 1. Return _quotient_.

Math.fmap ( _x_, _inLow_, _inHigh_, _outLow_, _outHigh_ )

When the `fmap` function is called, the following steps are taken:

1. If any argument is *NaN*, return *NaN*. 1. If _x_ is one of *+∞*, *-∞*, return _x_. 1. Let _dividend_ be ((_x_ − _inLow_) × (_outHigh_ − _outLow_)). 1. Let _divisor_ be ((_inHigh_ − _inLow_) + _outLow_). 1. Let _quotient_ be (_dividend_ ÷ _divisor_). 1. Let _x32_ be the result of converting _quotient_ to a value in IEEE 754-2008 binary32 format using roundTiesToEven. 1. Let _x64_ be the result of converting _x32_ to a value in IEEE 754-2008 binary64 format. 1. Return the ECMAScript Number value corresponding to _x64_.

Math.radians ( _degrees_ )

When the `radians` function is called, the following steps are taken:

1. If _degrees_ is one of *NaN*, *+∞*, *-∞*, return _degrees_. 1. Let _radians_ be (_degrees_ × `Math.RAD_TO_DEG`). 1. Return _radians_.

Math.degrees ( _radians_ )

When the `degrees` function is called, the following steps are taken:

1. If _radians_ is one of *NaN*, *+∞*, *-∞*, return _degrees_. 1. Let _degrees_ be (_radians_ × `Math.DEG_TO_RAD`). 1. Return _degrees_.
