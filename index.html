2Math.constrain ( x, lower, upper )#

When the constrain function is called, the following steps are taken:

  1. If any argument is NaN, return NaN.
  2. Let max be %Math_max%(x, lower).
  3. Let min be %Math_min%(max, upper).
  4. Return min.

3Math.map ( x, inLow, inHigh, outLow, outHigh )#

When the map function is called, the following steps are taken:

  1. If any argument is NaN, return NaN.
  2. If x is one of +∞, -∞, return x.
  3. Let dividend be ((x − inLow) × (outHigh − outLow)).
  4. Let divisor be ((inHigh − inLow) + outLow).
  5. Let quotient be (dividend ÷ divisor).
  6. Return quotient.

4Math.fmap ( x, inLow, inHigh, outLow, outHigh )#

When the fmap function is called, the following steps are taken:

  1. If any argument is NaN, return NaN.
  2. If x is one of +∞, -∞, return x.
  3. Let dividend be ((x − inLow) × (outHigh − outLow)).
  4. Let divisor be ((inHigh − inLow) + outLow).
  5. Let quotient be (dividend ÷ divisor).
  6. Let x32 be the result of converting quotient to a value in IEEE 754-2008 binary32 format using roundTiesToEven.
  7. Let x64 be the result of converting x32 to a value in IEEE 754-2008 binary64 format.
  8. Return the ECMAScript Number value corresponding to x64.

5Math.radians ( degrees )#

When the radians function is called, the following steps are taken:

  1. If degrees is one of NaN, +∞, -∞, return degrees.
  2. Let radians be (degrees × Math.DEG_TO_RAD).
  3. Return radians.

6Math.degrees ( radians )#

When the degrees function is called, the following steps are taken:

  1. If radians is one of NaN, +∞, -∞, return degrees.
  2. Let degrees be (radians × Math.RAD_TO_DEG).
  3. Return degrees.
